# TaskApp

概要:タスクと期限を入力できるメモアプリです。

使用技術:UserDefaults

学んだこと:

・Codableを使ってデータモデルをUserDefaultsに保存する

・シングルトンパターン

・テーブルビューセルの移動と削除

・UIAlertControllerの使い方

・UIToolbarの使い方

・UITextViewの使い方

・UITabBarControllerの使い方
